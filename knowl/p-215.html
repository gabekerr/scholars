Let \(A\) be an \(m \times n\) matrix in reduced row echelon form with \(r\) free variables with indices \(j_1 \lt j_2 \lt \cdots \lt j_r\) and \(s\) basic variables with indices \(i_1 \lt i_2 \lt \cdots \lt i_{s}\text{.}\) Then for the equation
\begin{equation*} A \mb{x} = \mb{b}, \end{equation*}
  1. There are no solutions if and only if \(s \lt m\) and \(b_i \ne 0\) for some \(s \lt i \leq m\text{.}\)
  2. If we are not in case (1), the vector
    \begin{equation*} \mb{x} = b_1 \mb{e}_{i_1} + b_2 \mb{e}_{i_2} + \cdots + b_s \mb{e}_{i_s} \end{equation*}
    is a solution and is unique if \(r = 0\text{.}\)
  3. If we are not in case (1) and \(r \gt 0\text{,}\) then for any \(1 \leq i \leq n\) define the function \(x_i : K^r \to K\) by
    \begin{equation*} x_i \left( \left[ \begin{matrix} t_1 \\ t_2 \\ \vdots \\ t_r \end{matrix} \right] \right) = \begin{cases} t_i \amp \text{ if } x_i \text{ is a free variable,} \\ b_i -a_{i j_1} t_1 - a_{i j_2} t_2 - \cdots - a_{i j_r} t_r \amp \text{ if } x_i \text{ is a basic variable.} \end{cases} \end{equation*}
    For \(\mb{t}\) in \(K^r\) take
    \begin{equation*} \mb{x} (\mb{t} ) = \left[ \begin{matrix} x_1 (\mb{t}) \\ x_2(\mb{t}) \\ \vdots \\ x_n (\mb{t}) \end{matrix} \right] . \end{equation*}
