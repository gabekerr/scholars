Lemma 2.7.17.

If \(B\) is a right inverse or left inverse of a square matrix \(A\text{,}\) then it is the inverse of \(A\text{.}\)

Proof.

Since \(A\) is a square matrix, it represents a linear transformation from \(K^n\) to \(K^n\text{.}\) If it has a left inverse \(B\) so that \(BA = I\) then that transformation is one-to-one. But by CorollaryÂ 2.6.5 we have that this means it is an isomorphism. Thus it has an inverse linear transformation which can be represented by a two sided matrix inverse \(A^{-1}\text{.}\) But then since \(BA = I\) we may multiply on the right by \(A^{-1}\) and obtain \(B = A^{-1}\text{.}\) A similar argument applies for the right inverse.
