Corollary 2.6.4.

Suppose \(U\) and \(V\) are finite dimensional vector spaces over \(K\) and \(T : U \to V\) is a linear transformation. Then
  1. if \(T\) is one-to-one then \(\dim (U) \leq \dim (V)\text{.}\)
  2. if \(T\) is onto then \(\dim (U) \geq \dim (V)\text{.}\)
  3. if \(T\) is a linear isomorphism then \(\dim (U) = \dim (V)\text{.}\)
