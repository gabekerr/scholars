For the second, notice that if \(E\) is Type I then \(EA\) is just \(A\) with two rows switched so by the alternating property and the first equation \(D(EA) = - D(A) = D(E) D(A)\text{.}\) On the other hand, if \(E\) is Type II so that \(E = E_i (c)\) then \(E A\) multiplies the \(i\)-th row by \(c\text{.}\) So row linearity and the second equation gives \(D(E A ) = c D(A) = D(E) D(A)\text{.}\) Finally, if \(E\) is Type III and equals \(E_{ij} (c)\) we have that the \(i\)-th row of \(E A\) is \(\mb{r}_i + c\mb{r}_j\text{.}\) So using row linearity, LemmaÂ 2.7.13, and the third equation we get
\begin{align*} D (EA) \amp = D(\mb{r}_1, \ldots, \mb{r}_i + c\mb{r}_j, \ldots , \mb{r}_j, \ldots, \mb{r}_n ), \\ \amp = D(\mb{r}_1, \ldots, \mb{r}_i , \ldots , \mb{r}_j, \ldots, \mb{r}_n ) + c D(\mb{r}_1, \ldots, \mb{r}_j, \ldots , \mb{r}_j, \ldots, \mb{r}_n ),\\ \amp = D(\mb{r}_1, \ldots, \mb{r}_i , \ldots , \mb{r}_j, \ldots, \mb{r}_n ), \\ \amp = D (A), \\ \amp = D(E) D(A). \end{align*}
