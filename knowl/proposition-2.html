Proposition 2.1.6.

Matrix arithmetic satisfies the properties
  1. The matrix with all entries equal to zero is called the zero matrix and is an additive identity for matrix addition. All matrices have an additive inverse given by \(-A = (-1)A\) where the right hand side is scalar multiplication. The identity matrices act as multiplicative identities.
  2. Both matrix addition and multiplication are associative operations when defined
    \begin{align*} (A + B) + C \amp = A + (B + C) \amp (AB)C \amp = A(BC). \end{align*}
  3. Matrix addition is commutative
    \begin{equation*} A + B = B + A, \end{equation*}
  4. The distributive property holds when the operations are defined
    \begin{align*} (A + B) C \amp = A C + BC \amp A(B + C) \amp = AB + AC. \end{align*}
