By the fact that a dual vector is a continuous function, a linear scalar function is a continuous map which implies that composing commutes with limits. But finding the tangent vector \(\mb{r}^\prime (a)\) is such a limit and we see that
\begin{equation*} \mb{v}^* ( \mb{r}^\prime (a) ) = \left. \frac{d}{dt}\right|_a \mb{v}^* ( \mb{r} (t)) . \end{equation*}
Thus, if \(\mb{r}\) is differentiable, then the left hand side exists which implies the right exists as well.
