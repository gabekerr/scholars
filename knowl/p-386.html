Given matrices \(A, B\) and \(C\) where the operations below are well defined (i.e. they have compatible sizes to add or multiply), the following properties hold:
  1. \((A + B)^T = A^T + B^T\text{,}\)
  2. \((AC)^T = C^T A^T\text{,}\)
  3. A square matrix \(A\) is invertible if and only if \(A^T\) is invertible. In this case \((A^T)^{-1} = (A^{-1})^T\text{.}\)
  4. \(\det (A^T) = \det (A)\text{.}\)
  5. \(\rk (A^T) = \rk (A)\text{.}\)
