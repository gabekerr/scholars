Definition 3.2.8.

If \(V\) is an inner product space and \(U\) is a subspace, we say that the orthogonal complement to \(U\) is the vector subspace
\begin{equation*} U^\perp = \left\{ \mb{v} : \left\langle \mb{v} , \mb{u} \right\rangle = 0 \text{ for all } \mb{u} \in U \right\} \end{equation*}
