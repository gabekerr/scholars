Lemma 5.2.3.

Suppose \(T\) is represented as a Jordan block \(J_{\lambda, m}\) with respect to the basis \(\mathcal{B} = \{\mb{v}_0, \mb{v}_1, \ldots, \mb{v}_{m - 1}\}\text{.}\) For any \(0 \leq k \leq m - 1\) let \(\mb{y}_k\) be the path defined by
\begin{equation*} \mb{y}_k (t) = \frac{t^k}{k!} \mb{v}_0 + \frac{t^{k - 1}}{(k - 1)!} \mb{v}_1 + \cdots + \frac{t^1}{1!} \mb{v}_{k - 1} + \mb{v}_k. \end{equation*}
Letting
\begin{equation*} \mb{x}_k (t) = e^{\lambda t} \mb{y}_k (t) \end{equation*}
the general solution to
\begin{equation*} \dot{\mb{x}} = T ( \mb{x} ) \end{equation*}
is
\begin{equation} \mb{x} (t) = C_0 \mb{x}_0 (t) + C_1 \mb{x}_1 (t) + \cdots + C_{m - 1} \mb{x}_{m - 1} (t) \tag{5.2.2} \end{equation}
for scalars \(C_0, C_1 , \ldots, C_{m - 1}\text{.}\) This is the unique solution with initial condition
\begin{equation*} \mb{x} (0) = \threevec{C_0}{\vdots}{C_{m - 1}}_{\mathcal{B}}. \end{equation*}

Proof.
