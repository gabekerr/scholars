If \(Q\) is a quadratic form on \(V\text{,}\) then for any basis \(\mathcal{B}\) of \(V\) there is a unique symmetric matrix \(A_\mathcal{B}\) for which
\begin{equation*} Q (\mb{v} ) = \left( \coord{\mb{v}}{\mathcal{B}} \right)^T A_{\mathcal{B}}\coord{\mb{v}}{\mathcal{B}}. \end{equation*}
Furthermore, if \(\mathcal{C}\) is another basis, then there is an invertible matrix \(P\) for which
\begin{equation} A_{\mathcal{C}} = P^T A_{\mathcal{B}} P.\tag{4.3.2} \end{equation}
